# Age Calculator App

A simple Android application to calculate age based on the entered date of birth.

## Features

*   Calculates the age in years, months, and days.
*   Provides the total number of years, months, weeks, and days since birth.
*   Shows the user'''s zodiac sign.
*   Simple and intuitive UI.

## How To Build

1.  Clone the repository.
2.  Open the project in Android Studio.
3.  Create a `keystore.properties` file in the root directory of the project with the following content:
    ```
    storeFile=keystore.jks
    storePassword=<your_store_password>
    keyAlias=<your_key_alias>
    keyPassword=<your_key_password>
    ```
4.  Build the project using Gradle.
5.  Run the app on an emulator or a physical device.

## Technologies Used

*   [Kotlin](https://kotlinlang.org/)
*   [Jetpack Compose](https://developer.android.com/jetpack/compose)
